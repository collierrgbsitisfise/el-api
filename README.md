# Start el-api localy (Ubuntu 16.0x)

***install node last LTS version of node***
```sh
sudo apt-get update
sudo apt-get install nodejs
```

***set env settings in /app/config/***
```sh
mongoDbUr...
etc....
```

***start the project***
```sh
npm i
npm run <env>
```
